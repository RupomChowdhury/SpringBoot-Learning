# SpringBootApp

This repository contains my learning codes and experiments with Spring & Spring Boot framework.

## Overview
This project is a playground for exploring Spring & Spring Boot features, including Dependency Injection, Bean configuration, and various starter modules.

## Technologies Used
- Java 25
- Spring Boot 4.0.1
- Spring Web
- Spring Data JPA
- H2 Database
- Lombok
- Maven
- ModelMapper & MapStruct
- Stream API
- Lambda
- ReflectionUtils, ResponseEntity<>, Optional<>
- Exception Handling

## Getting Started

### Prerequisites
- Java 25
- Maven

### Running the Application
1. Clone the repository.
2. Navigate to the project directory.
3. Run the application using the SpringBoot@MainClass as this repository contains multiple mini application.
